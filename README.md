# OAC AI Assistant Embedding Workshop

A comprehensive hands-on workshop for embedding the Oracle Analytics Cloud (OAC) AI Assistant into your web applications.

## What You'll Learn

- Embed the OAC AI Assistant into any web application with a single HTML file
- Understand both authentication models: browser-based (session) and token-based (OAuth 2.0)
- Configure IDCS confidential applications for secure token generation
- Set up Safe Domains, manage token lifecycle, and implement production patterns
- Build side-by-side dashboard + AI Assistant layouts
- Set up a local Python development environment for testing

## Prerequisites

- Oracle Analytics Cloud instance (March 2026 or later recommended)
- Google Chrome browser
- Python 3.x (for local testing server)
- Text editor (VS Code, Notepad++, etc.)
