Copy, Paste, Embed

Three code snippets to embed the OAC AI Assistant into your existing web application. Replace the placeholder values with your OAC instance details and you're live.

1

Add the embedding library to your <head>

This loads the OAC embedding framework from your instance. It does not conflict with other libraries.

<script
  src="https://<your-oac-host>/public/dv/v2/embedding/auto/embedding.js">
</script>
⚠️
Your data source must be indexed for search in OAC before the embedded AI Assistant can query it.
2

Add the AI Assistant tag to your page

Place this wherever you want the Assistant to appear. The green highlighted portion is the Object ID — replace it with your own value from the dataset's Inspect panel.

<!-- RPD Subject Area -->
<oracle-analytics-assistant
  src-data="[&quot;A - Sample Sales&quot;]">
</oracle-analytics-assistant>
<!-- File-based Dataset -->
<oracle-analytics-assistant
  src-data="[&quot;XSA('<owner>'.'<dataset-name>')&quot;]">
</oracle-analytics-assistant>
<!-- Local Subject Area (LSA) -->
<oracle-analytics-assistant
  src-data="[&quot;XSA('<GUID>'.'<lsa-name>')&quot;]">
</oracle-analytics-assistant>
💡
How to find the Object ID: In OAC, navigate to Data, locate your dataset or subject area, right-click and select Inspect. The Object ID field in the Inspect panel is the value you use as the green highlighted placeholder above.
3

Initialize (after the assistant tag)

This activates all OAC embedded components on the page. Place it after the <oracle-analytics-assistant> tag.

<script>
  oracle.oa.embedding.ready()
    .then(oApp => oApp.applyBindings());
</script>
⚠️
Safe Domains: Your hosting domain must be registered in OAC under Console → Administration → Safe Domains with the Embedding checkbox enabled. Without this, the Assistant will be blocked by CORS.
🔒
Authentication: The code above uses browser-based (session) auth — users must be logged into OAC. For token-based auth (no login required), see Security Deep-Dive.
📄
CSS tip: Add display: block; width: 100%; height: 100%; to the oracle-analytics-assistant element to control its size within your layout.
